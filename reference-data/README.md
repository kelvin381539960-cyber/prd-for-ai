# Reference Data

This directory stores current reusable configuration, mapping, dictionary, FAQ, country/region, and notification source tables.

Reference data is not a historical PRD and is not itself the AI-readable product fact layer. When a table contains confirmed runtime facts, summarize the applicable rules into `knowledge-base/` and cite the source file.

## Current directories

| Directory | Purpose |
|---|---|
| `countries/` | Current country / region / phone code reference data |
| `notification/` | Current system notification content source data |
| `faq/` | Current FAQ source data |
| `transaction/` | Current transaction status or mapping source data |
