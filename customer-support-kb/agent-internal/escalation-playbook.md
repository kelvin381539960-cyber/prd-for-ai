# 转人工与升级 Playbook

## 通用升级原则

当问题涉及账户安全、身份验证、资金、交易、卡、合规、风控或无法通过通用排查解决时，应转人工。

## 用户需要准备的信息

根据问题类型，引导用户准备：

- 注册邮箱或手机号
- 问题发生时间
- App 页面截图
- 页面错误提示
- 交易金额、币种、网络
- TxID / 交易哈希
- 卡交易时间、金额、商户名称
- KYC 页面提示

## 不得要求用户提供的信息

- 密码
- 验证码
- 私钥
- 助记词
- 完整银行卡号
- 其他可用于接管账户的敏感信息

## 标准升级口径

```text
这个问题可能需要结合你的账户或交易信息进一步核查。为了保护你的账户安全，建议你联系人工客服协助处理。
```

## 必须升级的问题类型

- 账户无法登录且涉及安全风险。
- KYC 失败、长时间审核中或无法重新提交。
- 充值不到账、资产余额异常、交易状态异常。
- 卡被冻结、交易失败、重复扣款或金额异常。
- 用户要求追回、撤销、赔偿或责任判断。
- 涉及合规、地区限制、风控判断。
