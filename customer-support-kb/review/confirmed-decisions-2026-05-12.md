# 业务确认记录 - 2026-05-12

本文件记录用户在 2026-05-12 直接确认的客服知识库规则。

这些确认可用于回填 `customer-support-kb/`，但仍需在具体文件中根据内容范围选择 `partial_confirmed` 或 `confirmed`。

## Registration / Login

- 注册当前仅支持邮箱注册，不支持手机号注册；手机号注册以后支持，但不承诺时间。
- 登录支持邮箱、手机号和生物识别；生物识别即 Quick Login。
- 验证码有效期为 5 分钟。
- 验证码重发间隔、错误次数、锁定时长等，不对用户说明具体规则；统一说按页面提示稍后重试。
- 登录密码规则可以对用户明确说明。
- 推荐码为选填；没有推荐码也可以继续注册。
- AIX Tag / X-Tag 存在，但非必填。
- 账户被锁定、关闭、限制等情况统一转人工。

## KYC / Identity Verification / Wallet Opening

- KYC 当前流程包括：选择国家/地区、协议确认、证件验证、人脸验证、地址证明、提交审核。
- KYC 审核最长 3 个工作日。
- 当前 KYC 仅支持 PH；以后会陆续开放其他国家/地区。
- 不支持的国家/地区进入 Waitlist。
- KYC 失败原因可以对用户说明。
- 用户身份信息填错后是否允许重新提交或修改，要看具体在哪个环节。
- 地址证明必须提交。
- 地址证明文件大小限制 16MB 有效。
- 地址证明签发时间要求为 6 个月内。
- 人脸验证失败次数、锁定规则、超时规则可以对用户说明；具体数值如未明确，应以页面提示为准。

## Wallet Asset / Deposit / Send / Swap

- 当前支持币种：FDUSD、USDC、USDT、WUSD。
- 当前支持网络：BASE、BSC、ETHEREUM、SOLANA。
- 钱包资产页只展示稳定币。
- 钱包总资产默认按 USD 估算。
- 充值到账时间可以说明，但要强调不准确、仅供参考：ETH 约 10-15 分钟；其他网络约 1-5 分钟。实际到账时间以链上状态和 App 页面为准。
- 充值支持地址充值和 WalletConnect / 第三方钱包连接充值。
- 充值的最小金额、单笔限额、手续费、链上确认数可以对用户说明；具体值以 App 页面展示为准。
- Send 功能当前不可用。
- 通过手机号、邮箱、X-Tag 找收款人当前不可用。
- Swap 功能当前不可用。
- Swap 币种互换当前不可用。
- 用户充错链、错地址、错币种时，标准口径是无法保证找回，并应转人工。

## Card

- 当前开放卡申请。
- 卡申请必须先完成 KYC / 钱包开户。
- 当前支持虚拟卡和实体卡。
- 虚拟卡自动激活。
- 实体卡需要手动激活。
- 当前卡支持全球大多数国家/地区；具体以 App 页面提示为准。
- 卡申请失败原因可以说明，但具体风控拒绝原因不能说。
- 卡费用可以对用户说明，包括年费、申请费、制卡费、邮寄费、交易费、FX 费用等；具体金额以 App 页面展示为准。
- 卡限额可以对用户说明；具体限额以 App 页面展示为准。
- 单个用户最多可申请 5 张卡。
- 卡申请需要人脸验证。
- 卡申请失败后如已扣费，原路返回。
- 支持卡冻结、关闭、补发、重开；注销卡和补发需要联系客服。
- 实体卡激活需要输入卡后四位并设置 PIN。
- PIN 为 6 位数。
- 卡交易失败、重复扣款、退款异常需要分场景口径，待用户进一步提供。
- 卡退款是否退回卡余额、FX 费用和 transaction fee 是否退，继续保留待确认。

## Transaction History

- 当前交易状态：pending、success、failed、refunded、cancelled、declined、under review 都有效。
- 状态的正式中文口径不确定，继续保留待确认。
- 交易记录聚合钱包交易、兑换记录和卡交易。
- 交易记录可查询多久不确定，继续保留待确认。
- 第一期交易记录只支持筛选，不支持搜索和导出。
- 卡退款里的 REFUND / REVERSAL 都对用户显示为退款。

## Notification / System Email

- 当前 App 通知设置支持 Push、Email、SMS。
- 通知类型、是否可关闭、官方邮件域名、系统邮件排查口径、可疑邮件处理等问题继续保留待确认。

## Website / Waitlist

- 官网 / Waitlist 后续问题全部不确定，继续保留待确认。
