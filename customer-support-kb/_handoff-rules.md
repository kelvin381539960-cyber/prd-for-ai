# 转人工规则

当用户问题涉及账户安全、身份、资金、交易、卡、合规或无法通过通用步骤解决时，智能客服应引导用户联系人工客服。

## 必须转人工的情况

### 账户与登录

- 用户多次尝试仍无法登录。
- 用户无法访问原邮箱或手机号。
- 用户怀疑账户被盗或存在异常登录。
- 页面提示账户受限、冻结或存在安全风险。

### KYC / 身份验证

- KYC 审核失败，用户需要具体原因。
- KYC 长时间无结果。
- 用户无法重新提交资料。
- 用户提交的信息有误，需要修改身份信息。
- 涉及国家/地区、合规、制裁、风险判断。

### 钱包 / 资产 / 充值 / 发送 / 兑换

- 充值不到账。
- 链上显示成功但 App 未到账。
- 资产余额异常。
- 用户发送资产后无法找回。
- 交易失败但资产状态异常。
- 涉及错误网络、错误地址、错误币种。
- 用户投诉资金损失。

### 卡

- 卡申请失败或长时间无结果。
- 卡被冻结、锁定或无法使用。
- 卡交易失败、重复扣款、扣款金额异常。
- 用户要求取消、补发、关闭或恢复卡。
- 涉及卡限额、费用、支持国家、商户限制等未确认问题。

### 交易记录

- 交易状态长时间停留在处理中。
- 用户看不到交易记录。
- 交易金额、时间、状态与用户预期不一致。
- 用户要求撤销、追回或赔偿。

### 通知 / 邮件

- 用户持续收不到验证码、系统邮件或安全通知。
- 用户怀疑通知内容异常或存在钓鱼风险。
- 用户需要修改通知相关账户信息。

## 标准转人工口径

```text
这个问题可能需要结合你的账户或交易信息进一步核查。为了保护你的账户安全，建议你联系人工客服协助处理。
```

## 需要用户准备的信息

根据问题类型，客服可引导用户准备：

- 注册邮箱或手机号
- App 页面截图
- 问题发生时间
- 交易类型、金额、币种、网络
- TxID / 交易哈希
- 卡交易时间、金额、商户名称
- KYC 页面提示信息

## 不应要求用户提供的信息

智能客服不得要求用户提供：

- 密码
- 完整银行卡号
- 完整身份证件号码，除非在官方安全流程中
- 私钥、助记词、验证码
- 任何可绕过账户安全的信息
