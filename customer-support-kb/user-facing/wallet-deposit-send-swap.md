# 充值、发送与兑换

## 适用问题

- 如何充值？
- 充值前要注意什么？
- 充值不到账怎么办？
- 如何发送资产？
- 发送失败怎么办？
- 如何兑换资产？
- 兑换处理中或失败怎么办？
- 选择错网络、地址或币种怎么办？

## 标准答案

### 如何充值？

你可以在 App 的钱包或资产页面选择 Deposit / 充值入口，并按照页面提示选择充值方式、资产和网络。充值前请务必确认币种、网络、地址和最低充值要求是否正确。

具体支持的币种、网络、充值方式、最小金额和限制，请以 App 页面展示为准。

### 充值前要注意什么？

充值前请重点确认：

1. 币种是否正确。
2. 网络是否正确。
3. 地址是否完整无误。
4. 是否满足页面显示的最小充值要求。
5. 如果使用第三方钱包，请确认第三方钱包内的网络和资产与 App 页面选择一致。

如果你不确定，请先不要转账，建议联系人工客服确认。

### 充值不到账怎么办？

请先确认你选择的币种、网络和充值地址是否正确，并查看链上交易是否已完成。如果链上已确认但 App 内长时间未到账，建议联系人工客服，并准备好交易哈希、充值币种、网络、金额和时间。

### 如何发送资产？

如果 App 当前提供 Send / 发送入口，你可以按照页面提示选择收款人、资产和金额，并在确认页核对信息后提交。

发送前请确认收款人信息和金额无误。转出后通常无法保证撤回或找回。

### 发送失败怎么办？

请先查看页面提示。常见原因可能包括余额不足、收款人信息不正确、验证未通过或系统暂时无法处理。

如果页面提示失败、处理中时间过长，或资产状态异常，建议联系人工客服。

### 如何兑换资产？

如果 App 当前提供 Swap / 兑换入口，你可以选择卖出资产、买入资产和金额，查看页面展示的汇率和预估到账金额后再确认。

兑换前请认真核对资产、金额和汇率。具体可兑换资产、汇率、费用和限制请以 App 页面为准。

### 兑换处理中或失败怎么办？

如果兑换显示处理中，请耐心等待页面状态更新。如果长时间没有变化，或页面显示兑换失败但资产状态异常，建议联系人工客服。

## 排查流程

1. 确认用户操作类型：充值、发送或兑换。
2. 如果是充值，确认币种、网络、地址、金额、最低充值要求和 TxID。
3. 如果是发送，确认收款人、资产、金额、余额和验证状态。
4. 如果是兑换，确认卖出资产、买入资产、金额、汇率页面和结果状态。
5. 引导用户查看 App 内交易记录状态。
6. 如链上已确认但 App 未到账，或资产状态异常，转人工。
7. 如用户选择错网络、地址或币种，不承诺找回，转人工。

## 转人工边界

以下情况建议转人工：

- 充值长时间不到账。
- 链上成功但 App 未到账。
- 用户疑似选错网络、地址或币种。
- 发送或兑换失败后资产状态异常。
- 交易长时间处理中。
- 用户要求追回、撤销或赔偿交易。
- 用户询问未确认的支持币种、支持链、费用、限额、汇率或到账时间。
