# 充值、发送与兑换

## 适用问题

- 如何充值？
- 当前支持哪些币种和网络？
- 充值多久到账？
- 充值支持哪些方式？
- 充值有哪些费用或限制？
- Send 功能当前是否可用？
- Swap 功能当前是否可用？
- 选择错网络、地址或币种怎么办？

## 标准答案

### 如何充值？

你可以在 App 的钱包或资产页面选择充值入口，并按照页面提示选择充值方式、资产和网络。充值前请务必确认币种、网络、地址和最低充值要求是否正确。

当前支持资产：FDUSD、USDC、USDT、WUSD。

当前支持网络：BASE、BSC、ETHEREUM、SOLANA。

### 充值支持哪些方式？

当前支持地址充值，以及 WalletConnect / 第三方钱包连接充值。具体入口和流程请以 App 页面展示为准。

### 充值前要注意什么？

充值前请重点确认：

1. 币种是否正确。
2. 网络是否正确。
3. 地址是否完整无误。
4. 是否满足页面显示的最小充值要求。
5. 如果使用第三方钱包，请确认第三方钱包内的网络和资产与 App 页面选择一致。

如果你不确定，请先不要转账，建议联系人工客服确认。

### 充值多久到账？

充值到账时间不完全固定，仅可作为参考：ETH 网络可能约 10-15 分钟，其他网络可能约 1-5 分钟。

实际到账时间会受到链上状态、网络拥堵和系统处理影响，请以链上状态和 App 页面展示为准。

### 充值有哪些费用或限制？

充值最小金额、单笔限额、手续费和链上确认数可以对用户说明，但具体数值请以 App 页面展示为准。

### Send 功能当前是否可用？

Send 功能当前不可用。

通过手机号、邮箱或 X-Tag 查找收款人的能力当前也不可用。

### Swap 功能当前是否可用？

Swap 功能当前不可用，因此暂不支持币种互换。

### 充错链、地址或币种怎么办？

如果你已经发起转账，请尽快联系人工客服，并准备好交易哈希、币种、网络、地址、金额和时间。

请注意，充错链、地址或币种时，无法保证找回。具体情况需要人工进一步核查。

## 排查流程

1. 确认用户操作类型：充值、Send 或 Swap。
2. 如果是充值，确认币种、网络、地址、金额、最低充值要求和 TxID。
3. 明确告知 Send 当前不可用，手机号/邮箱/X-Tag 找收款人当前不可用。
4. 明确告知 Swap 当前不可用。
5. 引导用户查看 App 内交易记录状态。
6. 如链上已确认但 App 未到账，或资产状态异常，转人工。
7. 如用户选择错网络、地址或币种，不承诺找回，转人工。

## 转人工边界

以下情况建议转人工：

- 充值长时间不到账。
- 链上成功但 App 未到账。
- 用户疑似选错网络、地址或币种。
- 资产状态异常。
- 用户要求追回、撤销或赔偿交易。
- 用户对费用、限额、确认数或到账时间有争议。
