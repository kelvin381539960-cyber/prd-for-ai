# APK 反向事实源知识库

本目录用于存放从现网上线客户端 APK 反向拆解得到的事实源文档。

## 当前事实源

- [AIX Pay 1.0.0_10](./aix-pay-1.0.0_10/_index.md)

## 使用原则

1. APK 是现网客户端事实源，优先级高于历史 PRD。
2. APK 可直接证明：页面存在、路由结构、客户端接口调用、前端字段校验、按钮状态、客户端状态展示、静态文案。
3. APK 不能单独证明：服务端最终规则、后台配置、风控规则、供应商侧规则、运营配置。
4. 所有从 APK 反推的内容必须标注事实等级。
5. 正式 PRD 引用本目录内容时，应保留来源说明和待确认项。

## 事实等级

| 等级 | 含义 | PRD 使用方式 |
| --- | --- | --- |
| A | APK 直接可见，如路由、接口路径、静态文案 | 可进入 PRD 正文或事实附录 |
| B | 从前端逻辑强推，如跳转、字段校验、状态展示 | 可进入 PRD，但需表述为“当前客户端表现” |
| C | 从上下文推测，如接口用途、业务含义 | 进入待确认项 |
| D | 服务端、风控、后台或供应商规则 | 不得仅凭 APK 写入规则 |
