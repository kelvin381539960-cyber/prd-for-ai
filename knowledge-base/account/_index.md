# Account 账户模块

## 模块概述

AIX Card 的用户账户体系，覆盖用户从进入APP到正常使用的全部身份流程。

## 功能清单

| 功能 | 文档 | 状态 | 说明 |
|------|------|------|------|
| 注册 | [registration.md](./registration.md) | released | 邮箱注册 + OTP验证 + 设置密码 + 设置X-Tag |
| 登录 | [login.md](./login.md) | released | 邮箱/手机登录 + 密码验证 + Biometric快捷登录 |
| 忘记密码 | [password-reset.md](./password-reset.md) | released | 邮箱重置 + 身份验证 + 重设密码 |

## 账户状态机

```
┌──────────┐   注册成功   ┌──────────┐
│  未注册   │ ──────────→ │  Active  │
└──────────┘              └────┬─────┘
                               │
              ┌────────────────┼────────────────┐
              │                │                │
              ▼                ▼                ▼
        ┌──────────┐    ┌──────────┐    ┌──────────┐
        │  Locked  │    │  Banned  │    │  Closed  │
        └────┬─────┘    └──────────┘    └──────────┘
             │                                
             │ 锁定到期/重置密码               
             ▼                                
        ┌──────────┐                          
        │  Active  │                          
        └──────────┘                          
```

### 状态说明

| 状态 | 定义 | 触发条件 | 限制 | 解除方式 |
|------|------|----------|------|----------|
| Active | 正常使用中 | 注册成功后 | 无 | - |
| Locked | 安全原因临时锁定 | 登录失败超过N次 | 不可登录 | 锁定时长到期自动解除；或用户通过"忘记密码"重置 |
| Banned | 账户被限制（可恢复） | 风控触发（一期不支持） | 不可登录 | 联系客服处理 |
| Closed | 账户注销（不可恢复） | 风控/客服操作（一期不支持） | 所有功能不可用 | 无法解除 |

## 全局账户规则

### UID 生成
- 服务端在用户注册成功后生成

### 昵称规则
- 注册成功后自动生成：随机4位英文 + 随机6位数字
- 用户可手动修改（见 ME 模块）

### 唯一性约束
- **邮箱**：全局唯一，不允许重复注册或绑定
- **手机号**：全局唯一，不允许重复注册或绑定

### 设备绑定策略
- 自动绑定：注册/登录成功后，系统自动绑定当前 Device ID
- 绑定数量：单账户最多绑定 1 个 Device ID
- 同时在线：最多允许 1 个设备
- BIO 前提：仅设备已绑定时，登录方可启用 Biometric
