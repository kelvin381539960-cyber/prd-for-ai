# 2. 需求背景

为配合公司新业务 ——AIX 加密货币交易功能卡的正式上线，现计划同步推出其专属官方网站。网站将采用简洁现代的设计风格，清晰呈现 AIX 的核心亮点与主要特色，同时重点凸显产品的安全性与便捷性，以此建立用户信任、提升品牌形象，最终为整体业务增长提供助力。
